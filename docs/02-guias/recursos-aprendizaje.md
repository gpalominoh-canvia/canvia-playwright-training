# 🎥 Recursos de aprendizaje

Selección de videos y material para reforzar la capacitación. Están agrupados en
dos bloques: **bases de programación** (lo que conviene dominar antes y durante)
y **Playwright por nivel** (alineado con el [sílabus](../00-inicio/silabus.md)).

> Los recursos están en español salvo que se indique 🇬🇧. Son material de apoyo:
> la fuente de verdad sigue siendo la [documentación oficial](https://playwright.dev)
> y las guías de este repositorio.

## 🧭 Orden sugerido

**JavaScript básico → Async/await y promesas → TypeScript → Git/GitHub + GitFlow → Playwright por nivel (1 → 6).**
